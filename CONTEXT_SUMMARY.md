# AI Knowledge Base - Context Summary

## О проекте
Это персональная база знаний для работы с LLM, содержащая:
- Системные роли и персоны
- Проверенные навыки и паттерны
- Рабочие процессы (workflows)
- Результаты экспериментов и сравнений моделей

## Как использовать этот файл
**Для нейросети:** Этот файл — первый пункт входа. Он дает понимание структуры и философии репозитория. После прочтения можно запрашивать конкретные файлы по категориям или использовать скрипт сборки контекста.

## Структура в двух словах
| Папка | Назначение | Пример использования |
|-------|------------|---------------------|
| `roles/` | Кто я/агент | "Действуй как Senior Architect" |
| `skills/` | Конкретные умения | "Примени навык рефакторинга" |
| `workflows/` | Последовательности | "Запусти код-ревью воркфлоу" |
| `experiments/` | Тесты гипотез | "Покажи результаты EXP-001" |
| `comparisons/` | Сравнения | "Сравни подходы из CMP-003" |
| `snippets/` | Кусочки промптов | "Вставь сниппет валидации" |

## Система тегирования
Все файлы имеют YAML frontmatter с полями:
- `type`: роль, навык, воркфлоу, эксперимент, сравнение, сниппет
- `tags`: произвольные теги (язык, задача, домен)
- `models`: для каких моделей работает (qwen, claude, gpt4, etc.)
- `status`: draft / verified / deprecated
- `experiment_id` / `comparison_id`: идентификаторы для группировки

## Быстрые команды
```bash
# Собрать весь контекст
python utils/build_context.py

# Фильтровать по статусу и тегам
python utils/build_context.py --status verified --tags python,testing

# Получить конкретный эксперимент
python utils/build_context.py --experiment EXP-001
```
