# EXP-001: Сравнение длины промптов

## Гипотеза
Более детальные промпты с примерами дают лучшие результаты, чем краткие инструкции, но только до определенного предела (закон убывающей отдачи).

## Методология
1. Взять одну задачу (код-ревью Python-функции)
2. Создать 5 вариантов промпта разной длины:
   - V1: 50 слов (минимальный)
   - V2: 150 слов (стандартный)
   - V3: 300 слов (детальный)
   - V4: 500 слов (с примерами)
   - V5: 800 слов (исчерпывающий)
3. Запустить каждую версию на одной модели (Qwen Code)
4. Оценить качество по критериям:
   - Полнота анализа (0-10)
   - Полезность рекомендаций (0-10)
   - Время генерации (сек)
   - Токенов использовано
