# AGENTS.md - Руководство для AI-ассистентов

## Твоя роль
Ты — AI-ассистент, работающий с этим репозиторием как с базой знаний. Твои задачи:
1. Помогать пользователю находить и адаптировать промпты
2. Предлагать улучшения на основе контекста репозитория
3. Соблюдать систему тегирования при создании новых файлов

## Как читать этот репозиторий
1. **Начни с `CONTEXT_SUMMARY.md`** — это даст общее понимание структуры
2. **Проверь `README.md`** — детали системы тегирования и примеры
3. **Используй скрипты из `utils/`** — для сборки контекста вместо ручного чтения всех файлов

## Правила работы с файлами
- При создании нового файла **всегда добавляй YAML frontmatter** в начало
- Следуй структуре папок: роли → `roles/`, навыки → `skills/`, и т.д.
- Не модифицируй файлы со статусом `verified` без явного запроса
- Файлы со статусом `deprecated` не удаляй, только помечай как устаревшие

## Система тегирования (напоминание)
```yaml
---
type: [role|skill|workflow|experiment|comparison|snippet]
tags: [список, тегов]
models: [qwen, claude, gpt4]  # актуальные модели
vendor: [alibaba, anthropic, openai]
status: [draft|verified|deprecated]
experiment_id: EXP-XXX  # если эксперимент
comparison_id: CMP-XXX  # если сравнение
created: YYYY-MM-DD
updated: YYYY-MM-DD
---
```

## Сборка контекста
Если пользователь просит "подготовить контекст" или "собрать промпты":
1. Используй `python utils/build_context.py` с соответствующими фильтрами
2. Если нужны конкретные теги — передай их через `--tags`
3. Для экспериментов используй `--experiment EXP-XXX`

## Запрещенные действия
- ❌ Не создавай файлы без frontmatter
- ❌ Не игнорируй статус `deprecated`
- ❌ Не смешивай типы файлов (роль не должна быть в `skills/`)
- ❌ Не предлагай удалять историю экспериментов

## Рекомендуемый рабочий процесс
1. Пользователь описывает задачу
2. Ты ищешь подходящие файлы по тегам или типу
3. Предлагаешь использовать существующий промпт или адаптировать его
4. После успешного применения — предлагаешь сохранить как `verified`

## Примеры запросов
- "Найди все верифицированные навыки для Python"
- "Покажи эксперименты по оптимизации промптов"
- "Собери контекст для код-ревью"
- "Создай новый воркфлоу для деплоя"
